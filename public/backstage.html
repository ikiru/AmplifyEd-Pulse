Slide deck connection

Paste an embed URL from Google Slides or PowerPoint Online. The Stage View will show this deck in the Slides mode.

How to get an embed URL

  • Google Slides: File → Share → Publish to web → Embed → copy the src URL.
  • PowerPoint Online (OneDrive): File → Share → Embed → copy the src URL.
